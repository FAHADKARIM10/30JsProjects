# String Transformer

A **modern, interactive web tool** that transforms any input text into multiple formats such as **Lower Case, Upper Case, Camel Case, Pascal Case, Snake Case, Kebab Case, and Trimmed text**.  
Built with **HTML, CSS, and JavaScript**, this project demonstrates advanced **frontend skills, typography, animations, and micro-interactions** — perfect for portfolios or LinkedIn showcases.

---

## 🌟 Features

- **Real-time transformations**: Updates outputs as you type.  
- **Seven text formats**:  
  - Lower Case  
  - Upper Case  
  - Camel Case  
  - Pascal Case  
  - Snake Case  
  - Kebab Case  
  - Trimmed text  
- **Professional UI/UX**:  
  - Glassmorphism style cards  
  - Smooth hover & scale animations  
  - Subtle fade-in transitions for outputs  
  - Responsive layout for mobile and desktop  
- **Advanced typography**: Modern, readable fonts with shadow and spacing enhancements.  
- **Copy-to-clipboard functionality**: Click any output to copy text easily (if enabled via JS).  
- **Production-ready styling**: Clean, professional gradient palette and interactive elements.

## 🛠 String Transformations Implemented

| Transformation | Description |
|----------------|-------------|
| Lower Case     | Converts all letters to lowercase. |
| Upper Case     | Converts all letters to uppercase. |
| Camel Case     | First word lowercase, subsequent words capitalized, joined together. |
| Pascal Case    | Capitalizes first letter of all words, joined together. |
| Snake Case     | Converts spaces to underscores (_) and lowercases all letters. |
| Kebab Case     | Converts spaces to hyphens (-) and lowercases all letters. |
| Trim           | Removes extra leading and trailing spaces. |
