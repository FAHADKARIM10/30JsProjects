📞 Telephone Number Converter

A simple and interactive Telephone Number Converter built with HTML, CSS, and JavaScript. This project automatically formats your phone numbers as you type, making them cleaner and easier to read.


⚡ Features

Real-time formatting as you type.

Automatically converts numbers into a standard telephone format.

Handles incomplete inputs gracefully.

Easy-to-read, interactive UI.


💻 How It Works

User types a number into the input field.

JavaScript listens for input events on the field.

When the number reaches a certain length, it automatically formats it (e.g., adds dashes, parentheses, or replaces values).

The formatted number is displayed in the input field.


🛠️ Technologies Used

HTML – Structure of the page

CSS – Styling and layout

JavaScript – Real-time number formatting
